TUTUP

Deskripsi Project

KRPAI adalah sebuah devisi untuk mengembangkan robot yang ditujukan untuk dapat menyelesaikan sebuah misi untuk memadamkan api. Robot harus bisa mendeteksi api yang terdapat pada salah satu ruangan yang berbentuk labirin dan diharapkan dapat kembali ke tempat asal tanpa memasuki rungan lainnya.
Kalkulator digital ini dibuat menggunakan bahasa pemrograman C#. Aplikasi development yang digunakan adalah Visual Studio untuk membentuk sebuah grafik visual sebagai interface pengguna. Dapat menyelesaikan multi operasi, desimal, persentase, invers, trigonometri, akar, pangkat, bahkan konversi biner.
Aplikasi Ekspedisi / pegiriman logistik adalah aplikasi untuk mengelola layanan pengantaran barang serta digunakan sebagai alat input output kedalam database layanan. Menggunakan Bahasa C# adn database sebagai penyimpanan data. Aplikasi development yang digunakan adalah Visual Studio.
Huraken r-bot store shop merupakan sebuah aplikasi berbasis MS.Access dimana aplikasi ini adalah interface dari input output database yang menggunakan MS.Access sebagai media penyimpanannya. Fitur yang terdapat pada aplikasi ini diantaranya adalah laporan harian, laporan per tanggal, produk terpopuler, dan fitur-fitur lainnya.
Penghitung linier merupakan aplikasi yang dibuat menggunakan bahasa Java dan menggunakan aplikasi development NetBeans. Aplikasi ini mampu meyelesaikan masalah kalkulasi nilai maksimasi dan minimasi berdasarkan batasan-batasan yang kita buat. terdapat 2 metode perhitungan yaitu metode Simpleks dan metode Dualitas.
Kuis Miliyoner merupakan aplikasi yang dibuat menggunakan bahasa Java dan menggunakan aplikasi development NetBeans. Aplikasi ini akan membawa pengguna untuk menebak beberapa pertanyaan. Pertanyaan akan diberikan secara acak dan akan berganti apabila dijawab dengan benar. Setiap level memiliki nominal hadiah yang berbeda.
Aplikasi Rental Mobil merupakan aplikasi pertama yang saya buat menggunakan bahasa C++. aplikasi ini dapat membuat nota layanan yang berupa sewa mobil dan carter perjalanan. Untuk sewa mobil akan dihitung biaya sewa berdasarkan waktu. Sedangkan untuk carter perjalanan akan dihitung biasa sewa berdasarkan jarak perjalanan.

Posisi Saya

KRI merupakan ajang kompetisi robot yang berfokus pada perkembangan teknologi robotika yang beroprasi di darat seperti halnya robot sepak bola, tari, pemadam api, transporter, bahkan robot yang digunakan untuk mempermudah kegiatan / pekerjaan manusia di masa yang akan datang.
KRPAI adalah salah satu cabang dari jenis KRI yang merupakan robot cerdas pemadam api. Robot ini dirancang untuk dapat melewati rintangan dan melaju cepat serta didukung dengan penanaman kecerdasan untuk efisiensi pencarian titik api. tidak sekedar mencari api, robot juga diharapkan dapat menemukan jalan kembali ke posisi awal start.
Software merupakan bagian yang saya kerjakan yaitu memprogram dan mem-flash Program tersebut kedalam ic robot. program ini berisi logika-logika dan cara penggunaa sensor agar robot dapat berjalan dan dapat menyelesasikan misinya. Penulisan program didasarkan dengan trial and error dari uji coba sebelumnya agar mendapatkan hasil yang maksimal.
